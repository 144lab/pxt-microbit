# Screenshot

Make an [Image](/reference/image/image) out of the current state of the [LED screen](/device/screen).

```sig
led.screenshot();
```

### Parameters

* none

### Returns

* an [Image](/reference/image/image) of what is currently visible on the [LED screen](/device/screen)
